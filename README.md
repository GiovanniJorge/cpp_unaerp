# Repositório GiovanniJorge/CPP

[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## Sobre

Este repositório reúne exercícios e projetos desenvolvidos em linguagem C++, com foco no aprendizado, prática e aprimoramento das técnicas de programação. Os exemplos servem como base didática para estudantes e entusiastas da linguagem, abordando conceitos fundamentais e resolução de problemas diversos.

## Estrutura dos diretórios

- [`Exercícios-Geral`](https://github.com/GiovanniJorge/CPP/tree/main/Exerc%C3%ADcios-Geral): Exemplos e exercícios abrangentes de C++.
- `.gitignore` e `.gitattributes`: Arquivos de configuração do repositório.
- [LICENSE (MIT)](https://github.com/GiovanniJorge/CPP/blob/main/LICENSE): Licença de uso do projeto.

## Como utilizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GiovanniJorge/CPP.git
   ```
2. Navegue pelo diretório e escolha o exemplo que deseja compilar.
3. Compile os arquivos utilizando o `g++`:
   ```bash
   g++ exemplo.cpp -o exemplo
   ./exemplo
   ```
   *Substitua `exemplo.cpp` pelo caminho do arquivo desejado.*
